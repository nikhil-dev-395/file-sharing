# File Sharing App

**Tech Stack**: EJS, Tailwind CSS, Express.js, Node.js, MongoDB

## Running the Application

### Development Command

To run both the Tailwind CSS and Node.js server simultaneously, use:

```bash
yarn dev
```

# .env file

```
PORT = your port | 3000
MONGO_URI = your mongodb url
```
